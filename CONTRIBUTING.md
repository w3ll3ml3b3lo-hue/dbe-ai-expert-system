# Contributing to DBE AI Expert System

Thank you for your interest in contributing to the DBE AI Expert System!

## Code of Conduct
This project follows standard professional conduct guidelines. Please be respectful and collaborative.

## How to Contribute
1.  **Fork the repository** (if external) or create a new branch for your feature/bugfix.
2.  **Make your changes**.
3.  **Ensure code quality**: Follow PEP8 for Python and standard formatting for other languages.
4.  **Write tests**: Include unit tests for new functionality.
5.  **Submit a Pull Request**: Provide a clear description of the changes and link to any related issues.

## Branching Strategy
- `main`: Production-ready code.
- `develop`: Integration branch for new features.
- `feature/*`: Specific feature development.
- `bugfix/*`: Bug fixes.
